## 1. What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create web pages.

HTML defines the structure of a webpage using tags.

```html
<h1>Hello World</h1>
<p>This is a paragraph.</p>
```

---

## 2. What is the difference between HTML and HTML5?

| HTML                       | HTML5                                     |
| -------------------------- | ----------------------------------------- |
| Older version              | Latest version                            |
| Limited multimedia support | Supports audio, video, canvas             |
| Fewer semantic tags        | New semantic tags like `header`, `footer` |

---

## 3. What is the basic structure of an HTML document?

```html
<!DOCTYPE html>
<html>
<head>
  <title>Page Title</title>
</head>
<body>
  Content here
</body>
</html>
```

* `head` contains metadata
* `body` contains visible content

---

## 4. What are HTML tags and elements?

A tag is the markup like `<p>`.
An element includes the opening tag, content, and closing tag.

```html
<p>Hello</p>
```

---

## 5. What is the difference between block and inline elements?

| Block Element       | Inline Element             |
| ------------------- | -------------------------- |
| Takes full width    | Takes only required width  |
| Starts on new line  | Does not start on new line |
| Example: `div`, `p` | Example: `span`, `a`       |

---

## 6. What are semantic elements in HTML5?

Semantic elements describe the meaning of content.

```html
<header></header>
<nav></nav>
<article></article>
<footer></footer>
```

Benefits:

* Better SEO
* Better accessibility

---

## 7. What is the difference between `<div>` and `<span>`?

| `<div>`                  | `<span>`                      |
| ------------------------ | ----------------------------- |
| Block element            | Inline element                |
| Used for larger sections | Used for small inline content |

---

## 8. What are HTML attributes?

Attributes provide extra information about an element.

```html
<img src="image.jpg" alt="Photo">
```

---

## 9. What is the difference between `id` and `class`?

| `id`            | `class`                     |
| --------------- | --------------------------- |
| Unique          | Can be reused               |
| One per element | Multiple elements can share |

---

## 10. What are data attributes?

Custom attributes beginning with `data-`.

```html
<div data-user-id="101"></div>
```

```js
element.dataset.userId
```

---

## 11. What is the purpose of the `doctype` declaration?

```html
<!DOCTYPE html>
```

It tells the browser to render the page in standards mode.

---

## 12. What are meta tags?

Meta tags provide information about the webpage.

```html
<meta charset="UTF-8">
<meta name="description" content="HTML Tutorial">
```

---

## 13. What is the viewport meta tag?

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

It makes the page responsive on mobile devices.

---

## 14. What is the difference between GET and POST in forms?

| GET               | POST                 |
| ----------------- | -------------------- |
| Data in URL       | Data in request body |
| Less secure       | More secure          |
| Used for fetching | Used for sending     |

---

## 15. How do you create a form in HTML?

```html
<form action="/submit" method="post">
  <input type="text">
</form>
```

* `text`
* `email`
* `password`
* `number`
* `date`
* `file`
* `range`

## 17. What is the difference between `readonly` and `disabled`?

| `readonly`             | `disabled`             |
| ---------------------- | ---------------------- |

| User cannot edit       | User cannot interact   |

<input id="email" type="email">

| `nav`     | Navigation links      |

## 40. What is the `contenteditable` attribute?

```html
<div contenteditable="true">Edit me</div>
```

Allows user to edit content directly.

---

## 41. What is localStorage and sessionStorage?

| localStorage            | sessionStorage          |
| ----------------------- | ----------------------- |
| Permanent until cleared | Cleared when tab closes |

---

## 42. What is the difference between cookies, localStorage, and sessionStorage?

| Feature        | Cookies | localStorage | sessionStorage |
| -------------- | ------- | ------------ | -------------- |
| Size           | 4KB     | 5MB+         | 5MB+           |
| Sent to server | Yes     | No           | No             |

---

## 43. What is the purpose of the `<canvas>` element?

Canvas is used for drawing graphics with JavaScript.

```html
<canvas id="myCanvas"></canvas>
```

---

## 44. What is the purpose of the `<svg>` element?

SVG is used to create scalable vector graphics.

```html
<svg width="100" height="100"></svg>
```

---

## 45. Canvas vs SVG

| Canvas           | SVG              |
| ---------------- | ---------------- |
| Pixel-based      | Vector-based     |
| Better for games | Better for icons |

---

## 46. What are Web Workers?

Web Workers allow background JavaScript execution.

```js
const worker = new Worker('worker.js');
```

---

## 47. What is the `defer` attribute in script tags?

```html
<script src="app.js" defer></script>
```

The script loads in parallel and runs after HTML parsing.

---

## 48. What is the `async` attribute in script tags?

```html
<script src="app.js" async></script>
```

The script loads in parallel and runs immediately.

---

## 49. What is the difference between `defer` and `async`?

| defer                       | async                          |
| --------------------------- | ------------------------------ |
| Executes after HTML parsing | Executes as soon as downloaded |
| Preserves order             | Does not preserve order        |

---

## 50. What is the purpose of the `<noscript>` tag?

```html
<noscript>Please enable JavaScript</noscript>
```

Shown when JavaScript is disabled.

---

## 51. What are custom elements in HTML?

Custom elements let you create your own HTML tags.

```js
customElements.define('my-card', MyCard);
```

---

## 52. What is Shadow DOM?

Shadow DOM encapsulates styles and markup.

Useful in Web Components.

---

## 53. What is the difference between HTMLCollection and NodeList?

| HTMLCollection     | NodeList               |
| ------------------ | ---------------------- |
| Live collection    | Static collection      |
| Only element nodes | Can include text nodes |

---

## 54. What are microdata and structured data in HTML?

Structured data helps search engines understand content.

```html
<div itemscope itemtype="https://schema.org/Person"></div>
```

---

## 55. What is SEO in HTML and how can HTML improve it?

SEO stands for Search Engine Optimization.

HTML helps by:

* Using semantic tags
* Adding title and meta description
* Using proper heading structure
* Adding alt text to images
